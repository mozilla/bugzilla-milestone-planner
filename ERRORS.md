# ERRORS.md

This file documents detected inconsistencies in the Bugzilla bug data.

**Note:** This content is generated dynamically by the Enterprise Project Planner application. The live version is displayed in the web UI under "Detected Inconsistencies (ERRORS.md)" and can be copied from there.

## How to Generate

1. Open `index.html` in a browser (or serve via `python3 -m http.server 8080`)
2. Wait for the application to fetch data from Bugzilla
3. Scroll to the "Detected Inconsistencies (ERRORS.md)" section
4. Copy the generated markdown content

## Detected Error Types

The application detects the following inconsistencies:

### Dependency Cycles
Bugs that form circular dependencies, which make scheduling impossible.

### Orphaned Dependencies
Dependencies that point to non-existent bug IDs.

### Duplicate Summaries
Multiple bugs with identical or nearly identical summary text.

### Missing Assignees
Bugs without an assigned engineer.

### Missing Sizes
Bugs without a `[size=x]` tag in the whiteboard field.
